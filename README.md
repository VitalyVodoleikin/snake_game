# 🎮 Snake Game (`snake_game`)

Классическая игра «Змейка» на Python с использованием библиотеки `pygame`.
Управляйте змейкой, собирайте еду и избегайте столкновений — цель игры: набрать как можно больше очков.

Реализация культовой игры **Змейка** на Python с использованием библиотеки `pygame`. Чистый код, понятная архитектура.
Есть возможность расширить функционал.

---

## ✨ Особенности проекта

- ✅ Классическая механика игры: управление движением с помощью стрелок, случайное появление пищи.
- 🧠 Минималистичный и читаемый код — удобно изучать и модифицировать.
- 🛠 Легко кастомизировать: цвета, размеры, логику, счётчики.
- 📦 Зависимости собраны в понятном виде.
- 🐍 Python 3 + pygame — кроссплатформенность (Windows/macOS/Linux).

---

## !!! Реализовано

- Простое и интуитивное управление стрелками.
- Динамическое увеличение длины змейки при поедании еды.

## 📜 Некоторые заметки

Для упаковки игры на pygame в .exe-файл был использован PyInstaller.

Команды в виртуальном окружении:

```
# bash
pip install pyinstaller
pyinstaller --onefile --windowed main.py
```

Пояснения по флагам:
- --onefile — всё упаковано в один exe-файл (так удобнее для раздачи).
- --windowed (или -w) — не будет появляться чёрное консольное окно (отсутствие консольного окна для игр это обычно лучше).
- main.py — имя главного файла с игрой, который нужно упаковать в исполняемый файл.
После выполнения команды в папке проекта появились подпапки build и dist. .exe-файл находится в подпапке dist.
Также .exe-файл продублирован в основную папку, который можно запустить и играть.
